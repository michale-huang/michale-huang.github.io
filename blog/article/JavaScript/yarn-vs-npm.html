# yarn vs npm

# yarn优势对比

  • 更快
    • 并行安装包：npm和yarn在执行包的安装时，都是执行一系列任务，而npm执行任务是同步，即必须执行完当前包的安装，才能继续下一个包的安装，而yarn执行安装任务则是异步的，可以并行安装多个包，省去等待时间。
    • 缓存：
  • 打印信息更简洁
  • 命令更语义化
    • yarn add xxx,同时添加进package.json
  • lock文件
    • 用于跟踪项目依赖，确保安装版本一致
    • npm v5之后也有相同的功能
