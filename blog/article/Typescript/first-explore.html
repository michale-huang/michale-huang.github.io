Experimental support for decorators is a feature that is subject to change in a future release. Set the 'experimentalDecorators' option to remove this warning. In VSCode, Go to File => Preferences => Settings (or Control+comma) and it will open the User Settings file. Add "javascript.implicitProjectConfig.experimentalDecorators": true to the file and it should fix it. in tsconfig.json

{
  "compilerOptions": {
    "experimentalDecorators": true,
    "allowJs": true
  }
}

# .d.ts文件

类型定义文件，typescript中有几类诸如string，number的类型，但有事需要定义自己的类型，就写在types里面，比如：

// 对象
type user = {
  name:string,
  age:number
}
// 方法
type sayHello = (name:string) => string
// 类
type chen = {
  name:string,
  say: (age:number) => string
}
